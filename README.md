# MAYA Infrastructure Tracker

עדכון זה מוסיף התראה פנימית לעובד כאשר מנהל משייך אליו פרויקט.

## מה חדש
- כאשר מנהל יוצר פרויקט ומשייך אותו לעובד — העובד מקבל התראה במערכת.
- כאשר מנהל עורך פרויקט ומשנה שיוך לעובד חדש — העובד החדש מקבל התראה במערכת.
- ההתראה מופיעה בפעמון ההתראות ובמסך ההתראות.
- אין צורך להריץ SQL חדש אם כבר הרצת את קובץ ההתראות הפנימיות בעבר.

npm install
npm run dev
